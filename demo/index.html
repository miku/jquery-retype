jQuery RETYPE

Retype is a JQuery plugin which lets you type in a html textarea in multiple languages or multiple writing systems without switching the input language on your operating system. It has been tested on Gecko- and Webkit-based Browsers.

The mapping is dependent of the input keyboard used, e.g. a mapping tailored for a german keyboard layout will not trigger as expected on serbian latin keyboard, or won't trigger at all on a cyrillic keyboard.
