RETYPE

0.1.4 — 2009–02–03 — source and wiki http://bitbucket.org/miku/jquery-retype/

Retype is a JQuery plugin which lets you type in a html textarea in multiple languages or multiple writing systems without switching the input language on your operating system. It has been tested on Gecko- and Webkit-based Browsers.

The first example shows a native german input (since I am typing on a german keyboard) side by side with a russian keyboard map, which is suitable for native german speakers. The second example shows a ROT13 substitution cypher along with normal input, some example for multi character input, and a dvorak simplified keyboard mapping. Another example (in german) can be found here.
